This program finds the shortest path from the first position to the second one, for the knight chess figure.

The chessboard coordinates are numbered from 0 to 7 for both verticies and edges.


Opening the console, you'll see the test cases which are as follows:

  1. knightMoves([1,1], [1,3]) - [1,1] is the starting position while [1,3] is the end position we're searching for
  2. knightMoves([1,1], [0,1])
  3. knightMoves([1,1], [7,7])
  4. knightMoves([5,7], [6,3])
  5. knightMoves([0,0], [7,7])
  6. knightMoves([7,7], [7,6])
